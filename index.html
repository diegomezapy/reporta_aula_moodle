Reporta Aula Moodle

Appweb para extraer evidencias de participacion desde Moodle, calcular riesgo academico con trazabilidad bayesiana y publicar reportes operativos.

GitHub Pages activo

Tablero

KPIs, filtros globales, tablas, dona de riesgo, dispersion actividad/evaluacion y priorizacion.

Modelo

Probabilidad bayesiana con prior, posterior, log LR, version del modelo y factores auditables.

Backend

La extraccion Moodle usa FastAPI/Python. Debe desplegarse en Render, Railway, Fly.io o VPS.

Estado de publicacion

Esta URL publica corrige el error de GitHub Pages. Para usar la extraccion real, configure una URL publica del backend FastAPI y pruebe la conexion desde aqui.

Desplegar en Render Repositorio
Frontend estatico
GitHub Pages no ejecuta Python. La pagina publica evita el 404 y sirve como punto de entrada mientras se despliega el servidor de la app.
